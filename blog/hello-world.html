I started this site as my pet project. This is a GitHub pages that render ember app.

So what is GitHub page. You can get detailed info here. Github gives every account with unique URL like (accountname.github.io). You can build your web application using that a repo with name same as your username. That repository will have your site code.

What are the options to code your github page

I know the following options. There may be lot

  • You can just put a simple html file that will have all the content. And to make more sophisticated you can include your own javascript file and css file to that html file.
  • This is the more Suggested option. jekyllrb. It is bundle of option to help you to build your site. You can spend more time on your content then to code
  • OR, You can use some web client frameworks like ember, angular.

I used ember to make this site. Here I think I can give some information that may help people, how to put an ember app into GitHub page. Build the ember application using the command ember build. Now you will have deployable ember application inside the dist folder. Copy all the content inside the dist folder to your master branch of the GitHub page repository.

If you are building your site in ember, It might be helpful to have all your static content inside the public directory of ember app, It would be more convenient to be organized into separate directories under public directory of ember application

Now your site will render in your GitHub page URL, You can map your GitHub page URL to your own domain name if you have one. My GitHub page URL is https://datasweeper.github.io But i have mapped this URL to my domain name that i have purchased in godaddy.com http://datasweeper.org

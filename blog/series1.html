1. Setting up Python3.x and virtual environment


1.1 Installing Python3.x

Check for Python3.x in the system by issuing the following command.

$ python3 -V

If you got the following output, then you already having python3 you can skip to setting up virtual environment section.

Output:

Python 3.5.2

If your don't get the above output, Then your system don't have the python3 installed. You have to install it manually.

$ sudo apt-get update $ sudo apt-get -y upgrade $ sudo apt-get install python3.6

1.2 Installing pip - A python package manager

$ sudo apt-get install -y python3-pip

1.3 Installing virtualenv - virtual environment

$ python3 -m pip install --user virtualenv

Now you will have virtual environment install, and all the command utilities of virtual environment.
Create virtual environment using the following command.

$ virtualenv blog-env
$ cd blog-env
$ source bin/activate

Now you are in the virtual environment. It won't have any other unwanted python package or anything. It is minimalist python setup to start with.

2. Installing django

Inside your blog-env, Issue the following command to install django.

$ pip install django
$ ipython

Inside ipython terminal

In [1]: import django
In [2]: django.VERSION
Out[2]: (2, 0, 2, 'final', 0)

To Go out of ipython terminal

Cntr + d

You have installed django.

3. Starting a project

To start a django project issue the following command.

$ django-admin startproject blogapp
$ cd blogapp
$ ./manage.py runserver

Connect to http://localhost:8000 in your browser, Your project is now running. Lets start a app. to Stop the server.

Cntr + c

4. Starting an application

A django project can have multiple application, right now start an simple appliaction using the below command

$ ./manage.py startapp blog
